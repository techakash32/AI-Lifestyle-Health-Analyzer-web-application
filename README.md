# 🧠 AI Lifestyle Health Analyzer

> A Flask-powered web application that uses **Linear Regression** to predict your **Work Productivity Score** based on daily lifestyle habits — sleep, screen time, stress, caffeine, and more.

|-------|-----------|------|--------|
| Sign in page | Stats + charts | Lifestyle inputs | Score + health tips |

---

## 🚀 Features

- **AI Prediction** — Trained Linear Regression model predicts Work Productivity Score (1–10)
- **Lifestyle Form** — Collects 11 lifestyle inputs with sliders and dropdowns
- **Smart Health Tips** — Personalized recommendations based on predicted score band (Low / Moderate / High)
- **Dashboard** — Weekly screen time trend and sleep vs stress charts (Chart.js)
- **Prediction History** — Full log of all predictions with timestamps
- **User Profile** — View average score, total predictions, and update account info
- **Collapsible Sidebar** — Responsive layout with persistent state
- **MySQL Integration** — Data loaded from `health_db` database via SQLAlchemy

## 🧬 Machine Learning

### Target Column
| Column | Type | Description |
|--------|------|-------------|
| `Work_Productivity_Score` | int (1–10) | **Prediction target** |

### Feature Columns
| Feature | Type | Description |
|---------|------|-------------|
| `Age` | int | User's age (18–60) |
| `Gender` | categorical | Male / Female / Other |
| `Occupation` | categorical | Professional / Student / Business Owner / Freelancer |
| `Device_Type` | categorical | Android / iOS |
| `Daily_Phone_Hours` | float | Average daily smartphone usage |
| `Social_Media_Hours` | float | Daily time on social platforms |
| `Sleep_Hours` | float | Hours slept per night |
| `Stress_Level` | int (1–10) | Self-reported stress level |
| `App_Usage_Count` | int | Number of apps used daily |
| `Caffeine_Intake_Cups` | int | Cups of coffee/tea per day |
| `Weekend_Screen_Time_Hours` | float | Screen hours on weekends |

### Model Pipeline
```
MySQL DB → Pandas DataFrame → LabelEncoder → Train/Test Split → LinearRegression → joblib.dump()
```

### Score Bands
| Score | Band | Color |
|-------|------|-------|
| 1 – 3 | ⚠️ Low | Red |
| 4 – 6 | 📊 Moderate | Amber |
| 7 – 10 | 🚀 High | Green |

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/ai-lifestyle-health-analyzer.git
cd ai-lifestyle-health-analyzer
```

### 2. Install dependencies
```bash
pip install flask scikit-learn pandas numpy pymysql sqlalchemy joblib
```

### 3. Set up MySQL database
```sql
CREATE DATABASE health_db;
USE health_db;
-- Import your dataset into a table called `health_data`
-- Required columns: User_ID, Age, Gender, Occupation, Device_Type,
--   Daily_Phone_Hours, Social_Media_Hours, Work_Productivity_Score,
--   Sleep_Hours, Stress_Level, App_Usage_Count,
--   Caffeine_Intake_Cups, Weekend_Screen_Time_Hours
```

Update the connection string in `main.py` if needed:
```python
engine = create_engine('mysql+pymysql://root:YOUR_PASSWORD@localhost/health_db')
```

### 4. Train the model
```bash
python main.py
```
This connects to MySQL, trains the Linear Regression model, and saves `productivity_model.pkl`.

### 5. Start the Flask server
```bash
python app.py
```

Open your browser at **http://127.0.0.1:5000**
